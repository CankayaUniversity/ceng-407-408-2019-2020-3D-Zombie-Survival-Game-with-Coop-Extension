## 3D Zombie Survival Game
***
<ul> 3D ZSG is a co-op survival game that is popular with many players nowadays. ZSG (Zombie Survival Game) has been designed by Çankaya University computer engineering students by Bilal BATUR, Kaan YILDIZ and Uğur Burak YAVUZ to be valid on macOS and Windows (above windows seven) platforms.</ul>
